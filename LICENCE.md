## 🧩 About / What Is This Tool?

**Fortnite Skin Changer** is a lightweight Windows application that allows you to change your in-game cosmetics — skins, backblings, pickaxes, gliders, and emotes [citation:2][citation:3][citation:10].

- **Skin Changer** — apply any skin instantly [citation:2]

## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\FortniteSkinChanger\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"F5"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"FortniteClient-Win64-Shipping.exe"` | Target process |
| `safe_mode` | Boolean | `true` | Restricts unstable features |
